# Restore Procedures

Team-internal procedures for restoring services during an incident.

## Documents

- `service-restore-template.md` - Template for service restoration
- `database-recovery.md` - Database restore procedure
- `verification-checklist.md` - Post-restore verification

## Template Structure

Each procedure should include:

1. **Prerequisites** - Access and tools needed
2. **Steps** - Numbered instructions
3. **Verification** - How to confirm success
4. **Rollback** - What to do if it fails
