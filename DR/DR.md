# DR routines review

On a recurring basis, we shall evaluate the need for DR improvements and and schedule any necessary work.

In this DR sync we ask the following questions:

- Have our DR processes been tested since previous -quarterly sync?
  - If not - schedule a test in upcoming quarter.  
  - If yes - continue
- Have we received any new requirements regarding DR?
  - If yes - discuss and write stories for implementing.  
  - If not - continue
- Have we found any improvements to the DR process?
  - If yes - discuss and write stories for implementing.  
  - If not - continue
- Have we reviewed our Backup configurations ensure that - backups are being taken in accordance with business - requirements?
  - If not schedule such a review
  - if yes - continue
- Has there been any disasters since previous quarterly s ync?
  - If yes - evaluate the DR routines with regards to this - happening.
- Do we need to make adjustments or are the routines fine?
